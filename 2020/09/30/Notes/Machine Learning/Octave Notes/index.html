Basic operations

基本运算 运算法则
+ 加法
- 减法
× 乘法
÷ 除法
^ 指数
sqrt() 根号
  • %表示注释
公式 作用
== equal
~= not equal
&& 与
|| 或
xor 异或

  • PS1('>>>') 将提示符 更改为>>>

  • 逻辑与：同真为真，否则为假

  • ; 句末输出分号，防止打印输出(supress the print output)

  • 输出一般变量，直接输入变量名即可

  • 对于复杂的变量，可以使用disp(a)来输出变量a，也可以使用C语言的旧式语法disp(sprintf('2 decimals:%0.2f',a))就输出保留两位小数后的变量ａ啦

  • format long 让字符串显示默认的位数

    　

Metrics

公式 作用
a=[1 2;3 4;5 6] 生成三行两列矩阵
a=[1;2;3] 生成向量
v=[1:0.1:2] 从1到2,步长为0.1的行向量
v=[1:6] 从1到6的行向量
ones(2,3) 两行三列全１矩阵
2*ones(2,3) 全2矩阵
zeros(1,3) 全0矩阵
rand(1,3) 随机矩阵(0~1)
w=randn(1,3) 高斯随机变量，服从正态分布
hist(w) 会绘制出w的直方图
hist(w,50) 横轴细分成50份
eye(4) 生成4x4的单位矩阵
help functio_name 帮助

  • 等价于第一条

  • v=[1;2;3] v=[1 2 3]

Move data around

公式 作用
size(A) 输出A的大小
size(A,k) 返回第k维度大小
length(A) 返回最大维度的大小
pwd octave当前路径
cd change directory（改变路径）
ls 列出当前文件夹的文件
load xxxx.xxx / load('xxxxx.xx') 加载数据
who 显示当前所有变量
whos 显示details
exit / quit 退出octave
clear A 清除A变量
v=A(1:10) 将A的前10个变量赋给v
save hello.mat v; 将变量v存储为hello.mat的文件
save hello.txt v -ascii 存储成文本文档或ascii编码的文件
A(3,2) A的第三行第二列
A(2,:) 输出A的第二行
A(:,2) 输出A的第二列
A([1,3],:) A中第一索引为1和3的所有元素
A=[A,[1;2;3]] 添加一列
A(:) 把A中所有元素放到一列
C=[A B] / C=[A,B] 把两个矩阵结合在一起（左右合并）
C=[A;B] 两个矩阵上下合并

  • 假如A是一个3×2的矩阵，那么size(A)返回的是一个1×2的矩阵

    A=[1 2;3 4;5 6]

  • size(A)

  • size(A,1)

  • length(A)

    但是length我们通常用在向量上，避免和矩阵弄混。比如length([1;2;3;4;5])

  • 单引号可以用来表示字符串

  • whos：

Calculate Data

公式 作用
A.*B element-wise 按元素乘
A*B 矩阵乘
A.^2 对每个元素求平方
1./A 对每个元素求倒数
log(A) 对每个元素求对数
-A 对每个元素求相反数
abs(A) 对每个元素求绝对值
A+ones(size(A,1),size(A,2),...,size(A,n)) 对每个元素加1
A+1 对每个元素加1
A' 对A求转置
max(A) 对矩阵A求最大元素的值
[val,ind]=max(A) 对矩阵A每一列求最大值
A<3 返回和A同等大小的矩阵，1表示真，0表示假
find(A) 返回比3小的索引
magic(n) 返回一个行、列、对角线之和都相等的n×n矩阵
[r,c]=find(A>=3) 返回符合条件所在的行和列
sum(A) 返回所有元素之和
prod(A) 返回所有元素之积
floor(A) 向下取整
ceil(A) 向上取整
max(rand(3),rand(3)) 在两个随机矩阵中取对应元素中较大的元素组成
max(A,[],1) 取每一列的最大值，1表示从第一维度取值
max(A,[],2) 取每一行的最大值，2表示从第二维度取值
max(max(A)) / max(A(:)) 矩阵A中的最大值
sum(A,1) 对第一维度（列）求和
flipud(A) 使矩阵垂直翻转
pinv(A) 求A的伪逆矩阵
pinv(A)*A 可以看到对角线近似为1

Plot Data

公式 作用
hold on 在旧的图像上绘制新的图像
plot(x,y,'r) 用红色的线绘制
xlabel('time') 横轴标签为time
ylabel('value') 纵轴坐标为value
legend('sin','cos) 第一条曲线标识为’sin’，第二条曲线标识为”cos”
title('my plot') 标题为my plot
print -dpng 'my plot.png' 保存为文件
cd 'path'; print -dpng 'my plot.png' 保存到指定位置
close 关闭图像
figure(1); plot(x,y) 为图像标号
subplot(1,2,1) 把图像分成1×2的部分，现在用的是第一个格子
axis([1 2 3 4]) x轴范围是1到2，y轴范围是3到4
clf 清楚图像 clear a figure
imagesc(A) 矩阵可视化成彩色格子
imagesc(A),colorbar,colormap() gray; colorbar颜色条 矩阵可视化成灰度图像，并且添加颜色条，实际上是三个命令
command_1,command_2,command_3 三个命令同时执行
command_1;command_2;command_3 不输出任何东西
